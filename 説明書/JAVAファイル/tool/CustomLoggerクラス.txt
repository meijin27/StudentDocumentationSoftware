CustomLogger クラスに関する説明書

1. 概要
CustomLoggerクラスは、カスタマイズされたロギングのためのユーティリティクラスです。このクラスはユーザーのホームディレクトリ内にStudentDocumentationSoftwareLogという名前のフォルダを作成し、その中にログファイルを生成する機能を提供します。

2. コード詳細
2.1. 変数
logger: クラスレベルでのJava Logger インスタンス。このインスタンスは初回の要求時に初期化され、その後は再利用されます。

2.2. メソッド
getLogger: このメソッドは、クラスオブジェクトを受け取り、カスタマイズされたLogger インスタンスを返します。ログはユーザーのホームディレクトリ内のStudentDocumentationSoftwareLogという名前のフォルダに保存されます。もしフォルダが存在しない場合は、新しく作成します。

3. 使用方法
ロガーオブジェクトを取得するには、以下の方法でgetLoggerメソッドを呼び出します:
Logger myLogger = CustomLogger.getLogger(YourClassName.class);

取得したロガーオブジェクトを使用して、ログメッセージを生成します:
myLogger.info("This is an info message.");

4. 特徴
シングルトンアプローチ: このクラスはシングルトンのアプローチを使用しており、単一のLogger インスタンスのみが生成されます。
カスタムフォーマッタ: ログメッセージはカスタムフォーマットで出力されます。具体的には、日付（yyyy-MM-dd HH:mm:ss形式）、ログレベル、発生源のクラス名、そしてログメッセージが含まれます。

5. 注意事項
ログの初期化中に何らかの問題が発生した場合、この問題はlogger自体で記録されます。この場合、詳細なエラー情報はデフォルトのロガーコンフィギュレーションによってハンドルされます。
このクラスは、同期化された方法でロガーオブジェクトを提供するため、マルチスレッド環境でも安全に使用することができます。
